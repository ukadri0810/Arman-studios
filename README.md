# Editorial Photographer Website

A premium, responsive one-page photography website.

## How to use
1. Open `index.html` in any browser.
2. Replace the studio name, contact details and WhatsApp number.
3. Replace the Unsplash image URLs with the photographer's real photographs.
4. Upload `index.html` to GitHub Pages, Netlify, Firebase Hosting or any standard web host.

## Important edits
- Search for `ARMAAN STUDIO` to change the brand.
- Search for `919876543210` to replace the WhatsApp number.
- Search for `hello@armaanstudio.com` to replace the email.
- Package prices are inside the `<option value="">` fields and checkbox values.
